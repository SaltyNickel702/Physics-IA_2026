### required installs
- Python from [python.org](https://www.python.org/downloads/)
- ``pip install opencv-python``
	- Video viewing library

### Functionality
- Put the video file you want to analyze into the root folder
- Run ``python3 main.py``
- Type the name of the video, including file extension
- Type the name of the output data
- After a few seconds, the program will finish and a file will be created named ``[your name]_RAW.csv``
	- This just contains where the tick is
